# HEARTBEAT.md - Periodic Checks

_Things to check when polled. Keep it small, keep it useful._

## Quick Checks (Do These Always)

- [ ] Gateway status: `openclaw gateway status`
- [ ] Git status: any uncommitted changes?
- [ ] Today's memory file exists

## Rotating Checks (Pick 1-2 Each Time)

- [ ] Email — any urgent unread?
- [ ] Calendar — anything in next 24h?
- [ ] Weather — relevant for human's day?
- [ ] Projects — any stalled tasks in brain/cortex.md?

## When to Reply HEARTBEAT_OK

- Late night (23:00-08:00) unless urgent
- Nothing new since last check
- You just checked <30 min ago
- No action needed

## When to Actually Do Something

- Uncommitted changes → commit and push
- Gateway down → restart it
- Urgent email/calendar item → alert human
- Been >8h since you spoke → say hello
