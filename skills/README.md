# Skills

Recommended skills to install for your OpenClaw instance.

## Essential Skills

Install these first:

```bash
# Self-improving agent system
clawhub install self-improving

# Semantic memory search
clawhub install semantic-memory

# BrainOS sync to GitHub
clawhub install brainos-sync

# Backup and recovery
clawhub install backup-recovery

# Discord command center
clawhub install discord-command-center
```

## Recommended Skills by Use Case

### For Developers
```bash
clawhub install code-reviewer
clawhub install fullstack-developer
clawhub install project-planner
```

### For Research
```bash
clawhub install deep-research
clawhub install data-analyst
```

### For Teams
```bash
clawhub install sprint-planner
clawhub install strategy-advisor
clawhub install decision-helper
```

### For UX Work
```bash
clawhub install ux-designer
```

## Finding More Skills

```bash
# Search for skills
clawhub search <keyword>

# List all available
clawhub list

# Get info about a skill
clawhub info <skill-name>
```

## ClawHub

- Website: https://clawhub.com
- CLI docs: `clawhub --help`
